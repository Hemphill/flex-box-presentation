What is Flexbox?

Flexbox, a W3C level 1 module, is the first widely adopted proper CSS layout mechanism.

In the past we had to rely on floats or tables, which were not designed for the task. This left us with buggy, verbose & hacky solutions.

Basics & Terminology

Flex Container

The parent element containing Flex Items. It is defined by the CSS display property. 

display: flex;or display: inline-flex;

Flex Container Properties

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

flex-direction

Determines the direction of the main-axis. It alters the position and direction of the flex items.

Values: row (default), row-reverse, column, or column-reverse.

.container { flex-direction: row; }

flex-wrap

By default, flex items will stay on the same line. With the ability to shrink or expand as the layout changes.

The flex-wrap property can change this behavior in a similar fashion to how the word-wrap property affects text.

Values: nowrap (default), wrap, or wrap-reverse.

.container { flex-wrap: nowrap; }

flex-flow

A shorthand property for flex-direction and flex-wrap.

Defaults: row, nowrap.

.container { flex-wrap: nowrap; }

Flex Item

The child element containing your content. It resides inside a flex container.

Flex Item Properties

  • flex-grow
  • flex-basis
  • flex-shrink
  • flex

flex-basis

Determines the size of a flex-item prior to the space being determined.

Accepts standard CSS lenght values (e.g. px, %, em, rem, & etc.), or a keyword (currently only auto).

The auto keyword uses the width or height property, or defaults to allowing the flex-grow to determine the length.

flex-shrink

The relative factor to shrink by when the flex-items are longer than the space allows.

Key: flex-shrink | flex-basis

1 | 1000px
2 | 1000px

Accepts whole numbers. Default value is 1.

.flex-item { flex-shrink: 1; }

flex

A short hand for flex-grow, flex-shrink, & flex-basis. It can be expressed five ways:

  1. flex-grow, flex-shrink, & flex-basis
  2. flex-grow & flex-shrink
  3. flex-grow & flex-basis
  4. flex-grow
  5. flex-basis
 
.flex-item { flex: 0 1 auto; }

Tip: Setting the flex property is preferred over individual properties, because it sets sensible defaults for your flex item.

Example

We are going to build a simple grid. With a Flex Container and Flex Items inside it.

A Flex Container

We are going to call our flex container .row using the default left to right direction. 

.row {
  display: flex;
  flex-wrap: wrap;
}

  ... flex items go here ...

Simple stuff, doesn't do much of anything on it's own, we need flex items inside it to act as columns in our row.

A Basic Flex Item

Here we make a basic column. Each flex item with the col class will be the same width as each other.

.col {
  flex-basis: 0;
  flex-grow: 1;
  max-width: 100%;
}

Learning Tip

Flexbox started way back in 2007-2009, it's gone through 3 major iterations, and the final form came about in 2012. Thus, you might run into outdated blog posts or other online resources.

TIP: look for display: flex; in articles about Flexbox. If you see display: box; or display: flexbox; the article is about the older versions of the spec which is very different than the final version.

Basics & Terminology

Flex Container

The parent element containing Flex Items. It is defined by the CSS display property. 

display: flex;or display: inline-flex;

Flex Container Properties

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

flex-direction

Determines the direction of the main-axis. It alters the position and direction of the flex items.

Values: row (default), row-reverse, column, or column-reverse.

.container { flex-direction: row; }

flex-wrap

By default, flex items will stay on the same line. With the ability to shrink or expand as the layout changes.

The flex-wrap property can change this behavior in a similar fashion to how the word-wrap property affects text.

Values: nowrap (default), wrap, or wrap-reverse.

.container { flex-wrap: nowrap; }

flex-flow

A shorthand property for flex-direction and flex-wrap.

Defaults: row, nowrap.

.container { flex-wrap: nowrap; }
